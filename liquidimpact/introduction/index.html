  1. 企业形象展示

网站设计

  1. UI设计

    1. 我们设计两个主题稿供参考选择
    2. 通过Canvas或者CSS3来制作主页面的动画效果

  2. Single Page Application

    1. Make a web application but not website
    2. 无刷新
    3. 书签式进入，URL可记录网站状态，有利于分发以及浏览器重定向
    4. 通过横向滑动或者竖向滑动增加用户的空间感觉

  3. 响应式页面设计

    网站可以自动适应当前屏幕的大小

  4. 离线依然可用

    当网站被第一次加载后，即便用户没有链接互联网，用户依然可以看到之前浏览的数据

  2. 小尺寸移动设备
