Best Practise

It is the goal of NowJS to make the shared namespaces as carefree and easy to use as possible. However, if your application is running slowly or you are coding with performance in mind, here are a few considerations.

  1. Client function rather than remote function call.

  2. Sync values using remote function calls rather than using synced variables.

  3. Be careful on the timing of synced object

  4. Circular references makes error!

  5. Functions in object will be removed when the object passed as argument in remote function call

  6. Avoid deeply nested structures inside the `now` namespace
