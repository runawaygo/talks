HTML5 History API - window.history

 

  1. Properties

    1. history.length
    2. history.state

  2. Action Methods

    1. history.forward()
    2. history.back()
    3. history.go([delta])

  3. history.pushState(data, title [, url ] )

    Pushes the given data onto the session history, with the given title, and, if provided, the given URL.

  4. history.replaceState(data, title [, url ] )

    Updates the current entry in the session history to have the given data, title, and, if provided, URL.

  5. window.onpopstate

    Trigger when the history.state changed
